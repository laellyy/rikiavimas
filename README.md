💗 Rikiavimo algoritmų analizė — kursinis
Šitas projektas yra mano kursinis darbas, kuriame lyginu du rikiavimo algoritmus:

Selection sort (paprastasis)

Merge sort (spartusis)

Tikslas — pažiūrėti, kaip jie elgiasi su skirtingais duomenų tipais ir dydžiais.

Sugeneruoja tris duomenų tipus:

jau surikiuotus

atvirkščiai surikiuotus

Testuoja tris masyvų dydžius:

50000

Paleidžia abu algoritmus:

matuoja laiką (mikrosekundėmis)
